# `@interviewos/ui`

Storybook-only UI package for atomic component examples.

## Scripts

- `pnpm --dir packages/ui storybook`
- `pnpm --dir packages/ui build`
- `pnpm --dir packages/ui lint`
- `pnpm --dir packages/ui typecheck`

## Layout

- `.storybook` contains the Storybook runtime config.
- `src/stories/atoms`, `molecules`, and `organisms` hold the component examples.
